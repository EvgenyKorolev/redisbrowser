# Redis браузер

Небольшая программа которая может подключиться к redis и отправлять команды. Результат тоже можно посмотреть. Вообще программа написана с целью знакомства с redis, так что практической пользы от неё ожидать не стоит.

## Как запустить

Выложен просто Qt проект, самый простой способ начать работать с ним - открыть файл post.pro в QtCreator.

### Необходимые компоненты

Для работы нужен компилятор поддерживающий с++ 11 или старше и библиотека Qt, модули core gui network. Я Использовал Qt 5.11, но думаю, ничто не помешает этому коду работать на Qt 4.8 и старше.
